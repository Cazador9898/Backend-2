# Lista técnica de verificación EFT

- [x] Dos microservicios independientes y ejecutables.
- [x] Comunicación entre microservicios al registrar ventas.
- [x] JWT con sesión stateless.
- [x] Autorización por roles ADMIN, CAJERO y CLIENTE.
- [x] BCrypt para contraseñas.
- [x] Protección de llamada interna por API key.
- [x] Validación de entradas.
- [x] Manejo centralizado de errores.
- [x] Transacción de venta y descuento de stock.
- [x] OpenAPI en ambos servicios.
- [x] HATEOAS en recursos principales y notificaciones.
- [x] Pruebas unitarias y de seguridad.
- [x] JaCoCo configurado.
- [x] Dockerfiles y Docker Compose.
